The Architecture

A single AMD MI300X with 192GB HBM3 easily hosts multiple LLMs simultaneously via Ollama and ROCm 7.2 (e.g., Llama 3.3 70B + Qwen 2.5 32B + Mistral Small).

Three independent Node.js agents point to these local Ollama endpoints. They coordinate complex workflows in real-time using the ACMI MCP Server backed by Upstash Redis. Cloud round-trips kill agent loops; ACMI + AMD keeps it all local and fast.

Get Started

Run the protocol on your own AMD hardware in 30 seconds.

# 1. Start your local Ollama on ROCm
ollama run llama3.3:70b

# 2. Boot the ACMI MCP server for your agent
export UPSTASH_REDIS_REST_URL="https://your-endpoint.upstash.io"
export UPSTASH_REDIS_REST_TOKEN="your-token"
npx @madezmedia/acmi-mcp
