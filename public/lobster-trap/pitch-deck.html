SECURITY DISCLOSURE · JANUARY 2026

In January 2026, security researchers disclosed that IBM Bob can be tricked into executing malware via prompt injection.

“The CLI is vulnerable to prompt injection attacks that allow malware execution. The agent fails to block process substitution using >(command).” — PromptArmor disclosure, Jan 2026

Bob is the first major AI coding assistant to be publicly exploited at scale. The exploit bypassed Bob's own "human-in-the-loop" approval gate.

promptarmor.com/resources/ibm-ai-bob-downloads-and-executes-malware
CVE Disclosure · Critical
IBM AI ("Bob") downloads and executes malware
Bob's CLI silently invokes shell command substitution when generating bash output. An attacker who controls any tool response, fetched URL, or file content that Bob reads can execute arbitrary code on the developer's machine — bypassing the human-approval gate by hiding execution inside $(…) and >(…) process substitution.
